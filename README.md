# Easy Poster

**java的awt绘制海报的简单实现**

## 快速上手

- 看效果请直接运行 PosterTest
- 自定义海报请按照content目录下的类对自身业务兼容
- 如果装饰器不满足你的需求, 请对decorators扩展
- 绘制请让设计给出类似蓝湖的像素图, 仿照PosterTest按着填一下就ok
